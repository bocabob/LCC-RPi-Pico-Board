# Raspberry Pi Pico for LCC
 KiCad OpenLCB / LCC RPi Pico motherboards
 - 3.3V and 5V switching power regulation
 - provision for onboard or plug attachment of EEPROM or FRAM NVM modules
 - can be configured to be powered from LCC, independently, or inject power to LCC
 - tested to work with OpenLCB Single Thread and OpenLcbClib libraries
 
 LCC-Pico-Node v2.6
 - MCP2518 and MCP2562 chips for CAN bus implementation of OpenLCB/LCC protocols
 - ports for four NeoPixel / WS28xx strings of LEDs
 - port for 10 pin IDC connection of GPIO / power for peripheral devices
 - two I2C ports
 

 LCC-Pico-Stepper v2.4 A & B versions
 - MCP2518 and MCP2562 chips (A) or MCP251862 chip (B)for CAN bus implementation of OpenLCB/LCC protocols
 - port for parallel connection to a touch display panel
 - A4988 stepper motor driver port
 - ports for 10 pin IDC connection of GPIO / power / stepper peripheral devices
 - one I2C port
 
 10 pin IDC connection breakout boards
 - for 10 pin IDC connection of GPIO / power / stepper peripheral devices
